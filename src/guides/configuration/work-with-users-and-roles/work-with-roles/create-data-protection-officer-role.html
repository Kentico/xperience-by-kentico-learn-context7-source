Not a fan of the video format? Here’re the steps you need to take to create a Data protection officer role in Xperience by Kentico.

  1. Go to the Role management application.
  2. Add a New role and fill in the information:
    1. Display name: Data Protection Officer
    2. Code name: DataProtectionOfficer
    3. Description: e.g., “The Data Protection Officer oversight the personal data of company staff, customers, providers, or any other individuals (data subjects). The role is responsible for informing the data subjects about their data protection rights and handling requests for data erasure from them.”
  3. Save to create the role.
  4. Go to the Permissions tab.
  5. Select Add permission set for each application:
    1. Data protection: View, Create, Update, Delete.
    2. Contact groups: View.
    3. Contact management: View, Update, Delete.
    4. Recipient list: View, Delete.
    5. Members: View, Update, Delete.
  6. Save the changes.
