Xperience by Kentico’s Continuous integration feature allows you to manage database objects through source control using the .NET Command Line Interface. 

While the process is described in the documentation, seeing a walkthrough is often helpful.

In this video, Sean Wright walks through the process of propagating data and schema changes from one developer instance of Xperience to another using continuous integration.
