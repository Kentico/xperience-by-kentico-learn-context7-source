Xperience enables users to create and edit certain types of content via graphical interfaces. Referred to as “builders” in the documentation, these UIs comprise various components provided by the developers. This section of the documentation discusses available builder functionality and the types of extensions developers can introduce.

Page Builder

Page Builder provides a user-friendly interface where users can manage page content using configurable widgets prepared by the developers.

Form Builder

Form Builder offers an intuitive graphical editor that allows content creators and marketers to create and edit forms. You can find it on the Form builder tab of the Forms application when editing a form.

Email Builder

Email Builder offers a user-friendly graphical editor that allows content creators and marketers to compose emails out of reusable components. You can access Email Builder within email channel applications, by creating an email with an Email Builder template.
